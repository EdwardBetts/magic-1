guess(*args) {|db| ...}

Creates magic database and yields it to the given block

Example
  Magic.guess(:mime) { |db| db.buffer("Magic® File™") }
  # => "text/plain; charset=utf-8"

[Source]

# File lib/magic.rb, line 73
    def guess(*args)
      db = Database.new(*args)
      result = yield(db)
      db.close
      result
    end
